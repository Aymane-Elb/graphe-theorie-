‚ö° Algorithm Overview

Prim's algorithm is a greedy algorithm that finds a minimum spanning tree for a weighted undirected graph.

  • Time Complexity: O(E log V) with binary heap
  • Space Complexity: O(V)
  • Type: Greedy Algorithm
  • Application: Network design, clustering, approximation algorithms
  • Key Insight: Always choose the minimum weight edge that connects a visited vertex to an unvisited vertex

üîç How It Works

The algorithm builds the minimum spanning tree one vertex at a time, starting from an arbitrary vertex.

  • Start with any vertex as the initial tree
  • Repeatedly add the minimum weight edge that connects the tree to a new vertex
  • Continue until all vertices are included
  • The result is a tree with minimum total weight
  • No cycles are ever created during the process
